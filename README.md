# RaycastingEngine

A c++ raycasting engine made with SFML

## Current State 
 
A simple raycasting engine including :

- Textured walls 
- Working doors
- Thin walls suprot
- Mouse control for camera
- Dynamic map loading (as json exported from Tiled map editor)
